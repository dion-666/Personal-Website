# Personal Website

Personal site project for Dion Journey, currently planned as static HTML, CSS, and a touch of vanilla JS.

## Current Status

- Planning document: `website-plan.md`
- Next steps: scaffold static site structure inside `zapping-zenith/`

## Repo Setup

```
git clone git@github.com:dion-666/Personal-Website.git
cd Personal-Website
```

## Planned Stack

- Semantic HTML pages with shared partials/includes
- Custom CSS (flexbox/grid, prefers-reduced-motion aware)
- Optional vanilla JS sprinkles for nav, theme toggle, and form handling
- Buttondown for newsletter, Formspree/Resend for contact form
- Plausible analytics, deployed via Vercel or static hosting of choice

See `website-plan.md` for full details on IA, content, features, and timeline.
